Calculation Parameter Type Groups¶

A Parameter Group is a collection of values of an enumerable calculation parameter type (for more information about calculation parameter types see Calculation Parameter Types chapter in the Layout Tool guide). A parameter group can be passed as a calculation parameter to calculations that accept group parameters.

Groups are usually used for grouping of multiple rows of a grid into a single row. However, the calculation type determines how it uses a parameter. Some calculation may use a group for filtering.

Refer to the connector calculation documentation for more details.

  • Name

    Required
    The name of a group.

  • Calculation Parameter Type

    Required
    Type of calculation parameter which group is associated with.

  • Values

    List of available values for selected Calculation Parameter Type where each value can be added to the group.
