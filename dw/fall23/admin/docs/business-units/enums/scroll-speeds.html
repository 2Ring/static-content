Scroll SpeedsÂ¶

Scroll Speeds are used in Tickers and Sequences. In a Ticker, the Scroll Speed defines the rate at which the Sources in a Ticker move from one side to another. In a Sequence, the Scroll Speed defines the rate at which the sources transition when an animation is enabled. Scroll Speed is increased by using higher numeric value.

  • Name

    Required
    The name of the scroll speed.

  • Speed

    Required
    Value of scroll speed.
