Notification ChannelsÂ¶

Notification Channel represents external system which supports receiving notifications.

The notification subsystem needs to be enabled, before notifications can be sent. Please reach out to support@2ring.com to enable it for your instance.

  • Channel type

    Required
    Type of a channel

  • Name

    Required
    The name of a channel.

  • Enabled

    Required
    Whether notifications can be sent to this channel.

  • Log successful requests

    Required
    Normally, only failed requests are logged. Turning this option on cause also successful request to be logged.

  • Settings

    Settings for Notification Channel are defined in its Channel Type (for more info see Notification Channel Type Settings).
    When new Channel is created all settings having initial value configured will be pre-filled with this value.
