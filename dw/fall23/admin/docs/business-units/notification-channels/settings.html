Notification Channel Type Settings¶

When new setting is created by the system administrator, it’s automatically added to each already existing channel of particular type. When new Channel is created it’s initialized with all settings defined in Channel Type.

Cisco Webex Teams¶

  • Access Token - DW notification channels are associated with specific bot. Bot access token has to be set as value of channel’s Access Token setting.

DW Layout Banner¶

  • DWUser - User, under which messages are sent. This user has to be an administrator of Business Units to which channel is supposed be able to send messages.

MMS EZ Texting¶

  • User - EZ Texting User Name.

  • Password - EZ Texting Password.

SMS - Bandwith¶

  • Account Id - Id of the user account.

  • Authentication Token - Base64 encoded string consisting of “Api Token:Api Secret”

  • Sender - One of your telephone numbers the message should come from. Must be in E.164 format, like +19195551212.

  • Application Id - The ID of the Application your Sender number is associated with in the Bandwidth Phone Number Dashboard.

SMS - EZ Texting¶

  • User - EZ Texting User Name.

  • Password - EZ Texting Password.

SMS - Red Oxygen¶

  • Account Id - This is your Red Oxygen Account ID, which uniquely identifies your account.<br>Example: CI00000001

  • Password - This is the password of your Red Oxygen user account.

  • Sender Email - This is the email address of your Red Oxygen user account. It is also the address that will receive replies to the sent SMS.

SMS - Trumpia¶

  • User Name - Your account username (this can be found in the Account Settings page after logging in.)

  • Api Key - Your API key (This can be found in the API Settings page after logging in.).<br>Example value: 123456789abc1011

SMS - Twilio¶

  • Account Sid - This is your Twilio Account ID, which uniquely identifies your account

  • Authentication Token - Base64 encoded string consisting of “Account Sid:Auth Token”)

  • Sender - The Twilio phone number, short code, or Messaging Service that sends this message. This must be a Twilio phone number that you own, formatted with a ‘+’ and country code, e.g. +16175551212 (E.164 format).

SMS - Webtext¶

  • Api Id - Id of the Webtext Api.

  • Api Password - Password used for accesing to Webtext Api.

  • Sender - One of your telephone numbers the message should come from.
