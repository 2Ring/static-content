Calculation Parameter Types¶

Calculation parameter types are types of enumerable parameters that are supported by source system. These parameter types can then be used in KPI/Grid calculation type definition.
For list of supported enumerable calculation parameter types see particular connector’s guide.

  • Name

    Required
    The name of calculation parameter type.

  • Metadata

    A user can view or edit a metadata.

  • Item Separator

    Required
    Character or text used to separate individual parameter values when they are represented in one string.

  • Name-Value Separator

    Required
    Character or text used to separate name and value of the parameter value in string representation.

    Item Separator and Name-Value Separator can not have the same value.

  • Capabilities

    Capabilities of the connector, defined during the installation.

    • Lock Values

      Read-only
      If enabled, administrator cannot add, delete or edit any calculation parameter type values.
