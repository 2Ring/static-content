Values¶

  • Name

    Required
    The name of a parameter value.

  • Value

    Required
    Value of a parameter value.

System Connectors periodically checks source systems for available values for parameter types. Found values are then automatically updated. In custom Connectors available values have to be inserted and managed manually.

For manual refreshing of values for parameter types:

By clicking on PULL VALUES, connector gets the latest available values from the source system, while refresh button refreshes the values from the DW database.

  1. Click on PULL VALUES button located in panel’s toolbar.

  2. The pop up will appear, informing that the request has been submitted. A user has to wait until next connector iteration, which usually takes a few seconds.

  3. Refresh the list of values by clicking on the refresh button.
