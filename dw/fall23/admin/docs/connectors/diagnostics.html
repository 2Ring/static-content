Diagnostics¶

2Ring DW monitors each installed Connector and checks whether it is operating correctly. Following diagnostic information is available:

  • Licenses Consumed

    Number of licenses currently being used by logged in contact center agents.

  • Iteration

    Connector runs in iterations. In one iteration all outdated calculations are recalculated. This group contains information about last Connector’s iteration.

    • Result

      Indicates whether Connector was successful in collecting data from the source, processing the data and sending it to DW database. Succeeded or Failed is displayed.

    • Error

      An error message indicating why the Connector is not working.

    • Iteration Date

      The time of last iteration start. Client PC time is displayed.

    • Duration

      A number of milliseconds iteration took.

  • Runtime Data

    Some connectors store runtime data, that can be be viewed and used for troubleshooting.

    • Info

      Runtime data stored by the connector.

    • Last Successful Write

      Date and time of the last successful data write.
