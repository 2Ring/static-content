Filters¶

This panel lists all available license filters for particular connector. License filter filters which source system data are being collected, processed and displayed by DW.

For example by providing list of allowed skill groups, only data for agents belonging to these groups are being processed which in turn lowers the number of needed DW licenses.

Any number of available filters can be applied at the same time. For list of all available filters see specific connector’s guide.

For each available filter these properties can be set:

  • Value

    Value which should be applied to specific filter. When not set, filter is not being applied.

  • Item Separator

    Required
    Character or text used to separate individual values in filter value.
