Each Grid Calculation Function has a set of parameters that are accepted by calculations of that type.

  • Name

    Required
    The name of a parameter.

  • Type

    Required
    Type of values this parameter is consuming. If one of enumerable types is selected Origin of Values has to be provided. For more information see Calculation Parameter Types chapter.

  • Metadata

    A user can view or edit a metadata.

  • Origin of Values

    Available options are:

    • Values - Only values defined in Values can be used as parameter values

    • Groups - Only groups defined in Groups can be used as parameter values

    • Mixed - Both values and groups can be used as parameter values

  • Columns

    A column, based on which the the values are filtered.

  • Auxiliary Grouping Columns

    Available only if a filtering column is selected
    A list of columns, which can be used as an auxiliary columns for a grouping purpose. These columns, in addition to a column based on which the values are grouped, should also contain the name of the group.
