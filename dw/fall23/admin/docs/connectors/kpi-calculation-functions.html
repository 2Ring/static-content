KPI Calculation Functions¶

KPI Calculation Function is a named method of calculating a KPI.

  • Name

    Required
    The name of KPI Calculation Function.

  • Display Name

    A name that appears as the segment title when the KPI calculation function is selected. If the display name is not filled, the title is automatically set from the KPI calculation function name in which all words are separated by a space and all non-alphanumeric characters are also replaced by a space.

  • Description

    A description of the KPI Calculation Functions which appears in dropdown tooltips in both Administration Application and Layout Editor.

  • Data Type

    Required
    Type of value produced by this calculation function.

    Available options are Numeric, Text, DateTime and TimeSpan

  • Default Settings

    A JSON field that contains a default settings for KPI calculation function based segments. These settings are applied when the KPI calculation function is selected. The first level contains a type of the source and the second level contains default settings for the associated type.
    All available settings are listed in the Default Settings - Schema chapter.

    Example :

    {
    "kpi": {
        "displayFormat": "{0:F1}%",
        "presentationType": "Semaphore",
        "displayRangeMaximum": 100,
        "goodValueMinimum": 70,
        "goodValueMaximum": 100
    },
    "kpiBarLineChartSeries": {
        "displayFormat": "{0:F1}%",
        "goodValueMinimum": 70,
        "goodValueMaximum": 100
    },
    "calendar": {
        "goodValueMinimum": 70,
        "goodValueMaximum": 100
    }
}
  • IsCumulative

    Required
    Whether values of this calculation function can only raise over the span of one day.

  • Metadata

    A user can view or edit a metadata used for calculation.

  • Depends On Grid Calculation Function

    Whether the KPI is natively calculated by connector or the value is extracted from a Grid calculation function data.

    • Function

      Required only if Depends On Grid Calculation Function is enabled
      Which Grid Calculation Function the KPI depends on.

    • Function’s column

      Required only if Depends On Grid Calculation Function is enabled
      Which column from the selected Grid Calculation Function the KPI depends on.

    • Aggregation Function

      Required only if Depends On Grid Calculation Function is enabled
      A method how to merge values, if the row filter returns multiple rows.

      Available options are Value, Sum, Count, Mean, Min and Max

  • Capabilities

    Capabilities of the connector, defined during the installation.

    • Lock Parameters

      Read-only
      If enabled, administrator cannot add, delete or edit any KPI calculation parameters.
