Parameters¶

Each KPI Calculation Function has a set of parameters that are accepted by calculations of that type.

  • Name

    Required
    The name of a parameter.

  • Type

    Required
    Type of values this parameter is consuming. If one of enumerable types is selected Origin of Values has to be provided. For more information see Calculation Parameter Types chapter.

  • Metadata

    A user can view or edit a metadata.

  • Origin of Values

    Available options are:

    • Values - Only values defined in Values can be used as parameter values

    • Mixed - Both values and groups can be used as parameter values

  • Depends On Grid Calculation Function’s Parameter

    Available only if “Depends On Grid Calculation Function” on KPI calculation function is enabled
    Grid Calculation Function’s Parameter on which the KPI calculation function’s parameter depends.

  • Is Row Filter

    Available only if “Depends On Grid Calculation Function” is enabled
    Whether selected parameter is used to identify row from dependent grid calculation function data where KPI calculation function value is located.

    • Depends On Grid Calculation Function’s Column

      Available only if “Is Row Filter” is enabled.
      Grid Calculation Function’s Column on which the KPI calculation function’s parameter depends.
