License¶

A valid license is required for all connectors supplied by 2Ring.

To obtain a new commercial license for 2Ring connectors please contact your system integrator (SI) or reach out to 2Ring directly via any of the contacts listed on our website at https://www.2Ring.com/contact.

If you are an existing customer and your installation is covered by any of our support packages, then the fastest way to handle any license changes is by opening a new RFC ticket in our case tracking system – https://itop.2ring.com. If you do not know your credentials, please contact us via email at support@2Ring.com.

In DW administration under Connectors tab is a list of installed connectors. Connector with some errors (for example a license is missing, expired) is displayed with red text color .

Figure 2: Connectors tab

Click the Connector Name, then click on LICENSE… section .

Figure 3: License tab

In this tab, information about Connector’s license are available:

  • Product Name

    The name of the product the license is issued to.

  • Valid For Versions

    Versions of DW the connector licensed is valid for.

  • Issued To

    Company to which the license is issued to.

  • Expiration

    Date the license expires. Value unlimited means license never expires.

  • Number Of Licenses

    Number of licenses that can be consumed by logged in agents. Exceeding this number will cause the connector to stop calculating data and an error message will be displayed.

Uploading License¶

Connector’s license can be uploaded by clicking UPLOAD button. In opened dialog file from file system can be selected in License File field or license content can be inserted into License XML field. These two options are exclusive.

Figure 1: License upload dialog

Figure 2: Successful license upload

If selected license is invalid error will be shown in dialog after pressing Upload button.

Figure 3: Invalid license

The connector license must match the product version of the connector. DW only compares fields that do not contain a star character. E.g. connector license version 8.*.*.* is a valid license for connector product version 8.1.0.0.

Once the license is uploaded, it may take up to a minute for the license to take effect for the connector.

License Validation¶

License validation is performed at the beginning of each connector iteration. If the number of consumed licenses exceeds the number of purchased licenses, a license error will appear in the clients and no values will be calculated.

The state of the license validation in your environment can be checked at any time. This allows you to proactively monitor the number of consumed licenses in real-time as well as the peak license consumption.

To check the license validation, go to the DW administration. On Connectors tab select the desired connector and then select DIAGNOSTICS… section.

Figure: License Validation

License validation properties:

  • Licenses Consumed – the number of licenses currently being used

  • Peak Licenses Consumed – the largest amount of licenses used at any given time

  • Peak Licenses Consumed Date – the date on which the peak license consumption happened

License Filters¶

In large contact centers where only a portion of agents requires 2Ring DW to display real-time data, it could be quite expensive to buy licenses for all the agents using the contact center. Therefore, a license filter can be used to apply connector licenses to a specific area of the contact center and thus limit the number of licenses required.

To apply a license filter, navigate to the Connectors tab in DW administration, choose the desired connector and go to its FILTERS… section. Type in a comma-separated list of values, that will be used to filter the license.
