Channel Configuration

Discord is using webhooks as a way of sending messages to a text channel without having to log in as a bot. Webhooks are associated with specific channel.

Creation of webhook in Discord application is described here.

DW Configuration

In DW, variable part of a webhook URL that starts after /webhooks/ is used as a recipient of a notification message.

For example, only the highlighted part of the following webhook URL can be used as a recipient.

https://discordapp.com/api/webhooks/720912854964498162/lVHxGGVww9XeeKKUFiozfhLVFhjWaa_Iuf6_pBXVJKrumFl0OTDHAyFHaZLxvb5skT

Format of a Discord message with all options described can be found here.
