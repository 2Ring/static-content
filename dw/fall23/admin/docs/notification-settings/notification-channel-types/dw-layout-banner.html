DW Layout BannerÂ¶

DW can send messages to Banners which have their External Id property set.

External Id has to be unique across the entire Business Unit.

Sending messages to Banners cross Business Unit is not supported.

Only plain text can be sent to Banners. Formatting of text is not supported. External id is used as recipient address.

Channel Type Configuration

Header field has to have value:

Content-Type: application/json
Signature: {{BodyHash}}

Body field has to have value:

{
    "Text": "{{MessageBody}}",
    "SendTo": "{{RecipientAddress}}",
    "RuleBusinessUnitName": "{{BusinessUnit}}",
    "AlertGeneratedAt_UtcTimeZone": "{{AlertGeneratedAt_UtcTimeZone:R}}",
    "ChannelId": "{{ChannelId}}"
}
