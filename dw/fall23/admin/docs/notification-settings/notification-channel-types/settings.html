Settings¶

When new setting is created it’s automatically added to each already existing channel of particular type. When new Channel is created it’s initialized with all settings defined in Channel Type. If Setting has initial value defined this value will be prefilled as value of a channel setting. Changing initial value doesn’t update value of settings of existing channels.

  • Display name

    Required
    The name of Channel Type Setting.

  • System name

    Required
    System name is used as a placeholder identification. Avoid using spaces and other special characters in value.

  • Type

    Required
    Type of settings is used to validate provided value and to display value in appropriate field control.

    Available options are Numeric, Boolean, Text, Password, Enum and Url

  • Enumeration Options

    Available only when “Enum” Type is selected.
    Available enumeration options. Each option is separated with comma (without additional spaces).

  • Encrypted

    Available only when “Numeric”, “Text” or “Password” Type is selected.
    Whether the setting value will be encrypted.

  • Clear Sensitive Data On Change

    Whether value of passwords and encrypted settings will be cleared when the value of this setting is changed.

  • Is Required

    Whether the setting value is required.

  • Initial Value

    If provided, each new channel of this type will be created with this prefilled value.

  • Description

    Description of channel type.

  • Order index

    Required
    Order index specify in which order associated settings in channel should be displayed.
