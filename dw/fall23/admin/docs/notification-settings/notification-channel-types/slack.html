Channel Configuration

Slack is using webhooks as a way of sending messages to a text channel without having to log in as a bot.

Creation of webhook in Slack application is described here (first three chapters).

DW Configuration

In DW, variable part of a webhook URL that starts after /services/ is used as a recipient of a notification message.

For example, only the highlighted part of the following webhook URL can be used as a recipient.

https://hooks.slack.com/services/T015J7M53C1/U71577UFA5B/t0fQkUEkRGyEIAght5YHszdL

Slack message format documentation with all options described can be found here with interactive message builder here.
