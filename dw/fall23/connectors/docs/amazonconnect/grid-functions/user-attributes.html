1.4. User Attributes¶

The User Attributes Grid Function displays static data for Agent in a tabular format.

1.4.1. Columns¶

  • Agent Id - the Id of the agent

  • Agent Id Quoted - Agent Id column formatted for use in dynamic filters

  • Agent Name – the name of the agent

  • Agent Name Quoted - Agent Name column formatted for use in dynamic filters

  • Assigned Queues - The list of queues assigned to the agent

  • Assigned Queues Quoted - Formatted representation of the assigned queues for use in dynamic filters

  • Assigned Routing Profile - The routing profile assigned to the agent

  • Assigned Routing Profile Quoted - Formatted representation of the assigned routing profile for use in dynamic filters

  • Agent Username - The username of the agent

  • Agent Username Quoted - Agent Username column formatted for use in dynamic filters

  • Email - The email address of the agent

  • Email Quoted - Agent Email column formatted for use in dynamic filters

  • First Name - The first name of the agent

  • First Name Quoted - Agent First Name column formatted for use in dynamic filters

  • Last Name - The last name of the agent

  • Last Name Quoted - Agent Last Name column formatted for use in dynamic filters

  • Mobile - The mobile number of the agent

  • Mobile Quoted - Agent Mobile column formatted for use in dynamic filters

1.4.2. Calculation Parameters¶

  • Agents - The list of agents

  • Queues - The list of queue names
