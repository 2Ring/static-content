Post-Installation Configuration¶

Instead of a database connection, the configuration.xml file is used for configuration. This file is located in the installation directory (where Connector.exe file is located). Open the file in any suitable XML editor and see the bullet list below for XML tag information.

  • TestTimeout – how often the connector checks if calls, Texts or Staffino data should be generated

  • NotReadyTimeAfterRONA

    • Min – the minimum time an agent spends in Not Ready - RNA

    • Max – the maximum time an agent spends in Not Ready - RNA

  • Voice
    • Chance

      • ACD – the likelihood of an ACD call being generated

      • Internal – the likelihood of an internal call being generated

      • Outbound – the likelihood of an outbound call being generated

      • Abandon – the likelihood of an ACD call ending up as abandon

      • Dequeued – the likelihood of an ACD call ending up as dequeued

      • Hold – the likelihood of an ACD call being put on hold

      • ShortAbandon – the likelihood of an ACD call ending up as short abandon, when the call should have been abandoned on agent, but none was ready

    • QueueTime – the maximum time a call will spent in a queue

    • OutboundCallDuration

      • Min – the minimum duration of an outbound call

      • Max – the maximum duration of an outbound call

    • InternalCallDuration

      • Min – the minimum duration of an internal call

      • Max – the maximum duration of an internal call

    • ReservedTime – if the reserved time is greater than 10, the agent will get put into the RNA state and the call requeued

      • Min – the minimum time an agent spends in the Reserved state when offered an ACD call

      • Max – the maximum time an agent spends in the Reserved state when offered an ACD call

    • TalkTimeBeforeHold – not used

      • Min

      • Max

    • TalkTimeAfterHold

      • Min – the minimum time an ACD call spends in progress after being held

      • Max – the maximum time an ACD call spends in progress after being held

    • HoldTime

      • Min – the minimum time an ACD call spends being held

      • Max – the maximum time an ACD call spends being held

    • TalkTime

      • Min – the minimum time an ACD call spends in progress

      • Max – the maximum time an ACD call spends in progress

    • WorkTime

      • Min – the minimum time an agent spends in the Wrap-Up (Work) state after an ACD call

      • Max – the maximum time an agent spends in the Wrap-Up (Work) state after an ACD call

    • Text – refers to chats and emails

      • Chance – the likelihood of a Text being generated

      • AbandonChance – the likelihood of a Text ending up as abandon

      • ShortAbandonChance – the likelihood of an abandoned Text ending up as short abandon instead

      • DequeuedChance – the likelihood of a Text ending up as dequeued

      • QueueTime – the maximum time a Text will spent in a queue

      • TalkTime

        • Min – the minimum time a Text spends in progress

        • Max – the maximum time a Text spends in progress

      • ReservedTime – if the reserved time is greater than 10, the agent will get put into the RNA state and the Text requeued

        • Min – the minimum time an agent spends in the Reserved state when offered a Text

        • Max – the maximum time an agent spends in the Reserved state when offered a Text

    • Staffino

      • Chance – the likelihood of Staffino statistics increasing

      • ThankYousIncrease – the increase of ThankYous

      • SuggestionsIncrease – the increase of Suggestions

    • WrapUpCodes – a pool of wrap-up codes the connector chooses from

      • WrapUpCode – the name of the wrap-up code

    • CallVariables – a pool of call variable values the connector chooses from

      • CallVariable – the call variable value
