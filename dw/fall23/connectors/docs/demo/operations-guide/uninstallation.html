To uninstall the connector, follow these steps:

  1. Edit Connection.xml - see Connector Installation chapter for more details.

  2. Edit UninstallParameters.xml - change the value of DWDatabase parameter to the name of the DW database

  3. Edit UninstallService.cmd - change the value of servicename parameter to the name of the connector service

  4. Run UninstallService.cmd

  5. Run Uninstall.cmd - Enter the name of the DW database if prompted
