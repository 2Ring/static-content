Upgrade¶

DW Connector upgrade is always part of a bigger process described in the DW Deployment Guide, only follow the steps below as a part of that process:

  1. Edit Connection.xml - see Connector Installation chapter for more details.

  2. Edit UpgradeParameters.xml - DWDatabase – the name of the DW database

  3. Run Upgrade.cmd

  4. Run InstallService.cmd

Re-registration¶

When performing an upgrade of the DW application only, the connector has to be re-registered after the upgrade should the name of the DW database change. To re-register the connector, follow these steps:

  1. Edit Connection.xml - see Connector Installation chapter for more details.

  2. Run Register.cmd - Enter the name of the DW database once prompted

Documentation Update

In case of installing, or upgrading a connector, which is a higher version than the currently installed version of 2Ring Dashboards and Wallboards, a manual documentation update is recommended.
To do that, open the documentation zip-file, which is provided with the connector, and extract the files in the DW deploy folder -> docs -> connectors. Replace the files in the destination, if prompted.
