2.7. Agent Group Statistics¶

This Grid function pulls data from the Agent Group Statistics statistics type (data source) in Five9. The name after the ‘-‘ refers to the actual column name of the statistic type in Five9. For more information about the statistics consult an official Five9 documentation describing statistics types and their columns.

If a Derived column for a statistic is not specified, then it is false by default.

2.7.1. Columns¶

  • Agent Group Id - Agent Group Id

  • Agent Group Name - Agent Group Name

  • Call Charges - Call Charges

  • First Call Resolution - First Call Resolution

  • Inbound - Inbound

  • Internal - Internal

  • Internal Success - Internal Success

  • Manual - Manual

  • Not Ready Count - Not Ready Count

  • Outbound - Outbound

  • Preview - Preview

  • Preview Time - Preview Time

  • Processed VM - Processed VM

  • Skipped In Preview - Skipped In Preview

  • Total Calls - Total Calls

  • Total Calls Without Internal - Total Calls Without Internal

2.7.2. Calculation Parameters¶

The grid function has these optional parameters:

  • Agent Groups - Allows you to select from a list of Agent groups.
