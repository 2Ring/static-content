2.35. User Attributes¶

The User Attributes Grid Function displays static data for Agent in a tabular format.

2.35.1. Columns¶

  • Agent Id - Agent Id.

  • Agent Name - Agent Name.

  • Assigned Agent Groups - Assigned Agent Groups.

  • Assigned Skills - Assigned Skills.

  • Email - the email of the Agent.

  • Extension - the extension of the Agent.

  • Username - the username of the Agent.

  • Agent Id Quoted - Agent Id column formatted for use in dynamic filters.

  • Agent Name Quoted - Agent Name column formatted for use in dynamic filters.

  • Assigned Agent Groups Quoted - Assigned Agent Groups column formatted for use in dynamic filters.

  • Assigned Skills Quoted - Assigned Skills column formatted for use in dynamic filters.

  • Email Quoted - Email column formatted for use in dynamic filters.

  • Extension Quoted - Extension column formatted for use in dynamic filters.

  • Username Quoted - Username column formatted for use in dynamic filters.

2.35.2. Calculation Parameters¶

  • Agents - Allows you to select from a list of Agent names

  • AgentGroups - Allows you to select from a list of Agent groups.

  • Skills - Allows you to select from a list of skills using a logical OR operation for filtering. When you use this parameter, any object that matches at least one of the selected skills will be included in the results. - Allows you to select from a list of skills using a logical OR operation for filtering. When you use this parameter, any object that matches at least one of the selected skills will be included in the results.

  • SkillsAll - Allows you to select from a list of skills using a logical AND operation for filtering. When you use this parameter, only objects that match all of the selected skills will be included in the results. Please note that this parameter cannot be used simultaneously with ‘Skills’.
