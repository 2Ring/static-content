3.17. User Observations - Passive (UO_P)¶

This grid only supports the REALTIME interval.

3.17.1. Merge Columns¶

  • UserId - Agent’s user ID.

3.17.2. Columns¶

  • ActiveQueuesCount - Number of active queues.

  • MemberQueuesCount - Observed number of queues for a user.

  • UserName - Name of a user.

Because of request consumption, it is recommended to set a high refresh interval.

3.17.3. Calculation Parameters¶

The Grid function has these optional parameters:

  • Queue

  • User
