3.19. User Status Aggregates - Passive (USA_P)¶

This grid only supports the HISTORICAL interval.

3.19.1. Merge Columns¶

  • UserId - Agent’s user ID.

3.19.2. Columns¶

  • AgentRoutingCommunicatingTimeSum - Amount of time an agent spent in the Communicating status.

  • AgentRoutingIdleTimeSum - Amount of time an agent spent in the Idle status.

  • AgentRoutingInteractingTimeSum - Amount of time an agent spent in the Interacting status.

  • AgentRoutingNotRespondingTimeSum - Amount of time an agent spent in the Not Responding status.

  • SystemPresenceAvailableTimeSum - Total amount of time an agent spent in the Available status.

  • SystemPresenceAwayTimeSum - Total amount of time an agent spent in the Away status.

  • SystemPresenceBreakTimeSum - Total amount of time an agent spent in the Break status.

  • SystemPresenceBusyTimeSum - Total amount of time an agent spent in the Busy status.

  • SystemPresenceMealTimeSum - Total amount of time an agent spent in the Meal status.

  • SystemPresenceMeetingTimeSum - Total amount of time an agent spent in the Meeting status.

  • SystemPresenceOfflineTimeSum - Total amount of time an agent spent in the Offline status.

  • SystemPresenceOnQueueTimeSum - Total amount of time an agent spent in the On Queue status.

  • SystemPresenceTrainingTimeSum - Total amount of time an agent spent in the Training status.

  • UserName - Name of a user.

3.19.3. Calculation Parameters¶

The Grid function has these optional parameters:

  • Queue

  • User

  • UserGroup
