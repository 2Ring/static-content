2.2.1. AI_P_Department¶

Agent Info - Department - passively updated (produces more API requests for your organization)

This KPI function returns the department that a user belongs to.

Parameters: Queue, User

2.2.2. AI_P_Division¶

Agent Info - Division - passively updated (produces more API requests for your organization)

This KPI function returns the division that a user belongs to.

Parameters: Queue, User
