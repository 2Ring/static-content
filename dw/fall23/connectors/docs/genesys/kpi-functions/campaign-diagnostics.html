2.4. Campaign Diagnostics (CD_P)¶

2.4.1. CD_P_ActiveUsersInQueueCount¶

Campaign Diagnostics - Active Users In Queue Count - passively updated (produces more API requests for your organization)

This KPI function returns the number of users that are active in the queue.

Parameters: Campaign, Queue

2.4.2. CD_P_OutstandingInteractionsCount¶

Campaign Diagnostics - Outstanding Interactions Count - passively updated (produces more API requests for your organization)

This KPI function returns the number of currently ongoing interactions.

Parameters: Campaign, Queue

2.4.3. CD_P_ScheduledInteractionsCount¶

Campaign Diagnostics - Scheduled Interactions Count - passively updated (produces more API requests for your organization)

This KPI function returns the number of currently scheduled interactions.

Parameters: Campaign, Queue

2.4.4. CD_P_UsersInQueueCount¶

Campaign Diagnostics - Users In Queue Count - passively updated (produces more API requests for your organization)

This KPI function returns the number of users that are in the queue.

Parameters: Campaign, Queue

2.4.5. CD_P_UsersNotUtilizedCount¶

Campaign Diagnostics - Users Not Utilized Count - passively updated (produces more API requests for your organization)

This KPI function returns the number of users in the queue that are currently not engaged.

Parameters: Campaign, Queue

2.4.6. CD_P_UsersOnACampaignCallCount¶

Campaign Diagnostics - Users On A Campaign Call Count - passively updated (produces more API requests for your organization)

This KPI function returns the number of users currently engaged in a campaign call.

Parameters: Campaign, Queue

2.4.7. CD_P_UsersOnANonCampaignCallCount¶

Campaign Diagnostics - Users On A Non Campaign Call Count - passively updated (produces more API requests for your organization)

This KPI function returns the number of users currently engaged in communication that is not part of the campaign.

Parameters: Campaign, Queue

2.4.8. CD_P_UsersOnDifferentEdgeGroupCount¶

Campaign Diagnostics - Users On Different Edge Group Count - passively updated (produces more API requests for your organization)

This KPI function returns the number of users whose station is homed to an edge that is different from the campaign.

Parameters: Campaign, Queue

2.4.9. CD_P_UsersOnQueueCount¶

Campaign Diagnostics - Users On Queue Count - passively updated (produces more API requests for your organization)

This KPI function returns the number of users with the status of On Queue.

Parameters: Campaign, Queue

2.4.10. CD_P_UsersOnQueueWithStationCount¶

Campaign Diagnostics - Users On Queue With Station Count - passively updated (produces more API requests for your organization)

This KPI function returns the number of users that have a station and are in the queue.

Parameters: Campaign, Queue
