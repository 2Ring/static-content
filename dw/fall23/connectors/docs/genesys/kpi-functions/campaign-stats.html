2.6. Campaign Stats (CS_P)¶

2.6.1. CS_P_ContactRateAttemptsCount¶

Campaign Stats - Contact Rate Attempts Count - passively updated (produces more API requests for your organization)

This KPI function returns the number of call attempts made.

Parameters: Campaign, Queue

2.6.2. CS_P_ContactRateConnectsCount¶

Campaign Stats - Contact Rate Connects Count - passively updated (produces more API requests for your organization)

This KPI function returns the number of calls in which a live voice was detected.

Parameters: Campaign, Queue

2.6.3. CS_P_EffectiveIdleAgentsCount¶

Campaign Stats - Effective Idle Agents Count - passively updated (produces more API requests for your organization)

This KPI function returns the number of effective available agents that are not being utilized currently.

Parameters: Campaign, Queue

2.6.4. CS_P_IdleAgentsCount¶

Campaign Stats - Idle Agents Count - passively updated (produces more API requests for your organization)

This KPI function returns the number of agents who are on queue and are available to take interactions, but are not working with any interactions.

Parameters: Campaign, Queue

2.6.5. CS_P_OutstandingCallsCount¶

Campaign Stats - Outstanding Calls Count - passively updated (produces more API requests for your organization)

This KPI function returns the number of currently ongoing campaign calls.

Parameters: Campaign, Queue

2.6.6. CS_P_ScheduledCallsCount¶

Campaign Stats - Scheduled Calls Count - passively updated (produces more API requests for your organization)

This KPI function returns the number of currently scheduled campaign calls.

Parameters: Campaign, Queue
