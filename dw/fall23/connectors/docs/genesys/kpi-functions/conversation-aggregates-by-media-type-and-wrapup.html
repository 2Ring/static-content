2.8. Conversation Aggregates By Media Type And Wrap Up (CAByW_A)¶

2.8.1. CAByW_A_ACWCount¶

Conversation Aggregates By Media Type and Wrap Up - ACW Count - actively updated with notifications

This KPI function returns the number of times agents completed after-call work.

Parameters: MediaType, Queue, WrapUpCode

2.8.2. CAByW_A_ACWSum¶

Conversation Aggregates By Media Type and Wrap Up - ACW Sum - actively updated with notifications

This KPI function returns the total time spent in after-call work (ACW). After-call work is the work that an agent performs immediately following an interaction. Work may involve keying activity codes, updating customer databases, filling out forms, or initiating outbound contact. As a metric, it indicates the total amount of time an agent takes to finish working on a call. If an agent must complete any work before the agent can handle the next contact, then ACW is factored into the average handle time.

Parameters: MediaType, Queue, WrapUpCode

2.8.3. CAByW_A_HandleCount¶

Conversation Aggregates By Media Type and Wrap Up - Handle Count - actively updated with notifications

This KPI function returns the total number of ACD and non-ACD interactions agents were involved in, including answered interactions, transfers, callbacks, and outbound calls. Handle increments in the interval in which the interaction ends, after the agent completes any after-call work and completes the interaction.

Parameters: MediaType, Queue, WrapUpCode

2.8.4. CAByW_A_HandleSum¶

Conversation Aggregates By Media Type and Wrap Up - Handle Sum - actively updated with notifications

This KPI function returns the total amount of time that it takes to establish a connection with the station on an outbound call.

Parameters: MediaType, Queue, WrapUpCode

2.8.5. CAByW_A_HoldCount¶

Conversation Aggregates By Media Type and Wrap Up - Hold Count - actively updated with notifications

This KPI function returns the number of interactions in which the interaction went on hold.

Parameters: MediaType, Queue, WrapUpCode

2.8.6. CAByW_A_HoldSum¶

Conversation Aggregates By Media Type and Wrap Up - Hold Sum - actively updated with notifications

This KPI function returns the cumulative hold time for all interactions.

Parameters: MediaType, Queue, WrapUpCode

2.8.7. CAByW_A_TalkCount¶

Conversation Aggregates By Media Type and Wrap Up - Talk Count - actively updated with notifications

This KPI function returns the number of times an agent was talking/interacting.

Parameters: MediaType, Queue, WrapUpCode

2.8.8. CAByW_A_TalkSum¶

Conversation Aggregates By Media Type and Wrap Up - Talk Sum - actively updated with notifications

This KPI function returns the amount of time spent interacting with a type of media.

Parameters: MediaType, Queue, WrapUpCode

2.8.9. CAByW_A_TransferredCount¶

Conversation Aggregates By Media Type and Wrap Up - Transferred Count - actively updated with notifications

This KPI function returns the number of interactions an agent answered in one queue and then transferred. A transfer counts for the queue in which an agent answers the interaction. This measurement is calculated by the number of interactions blind transferred or consult transferred.

Parameters: MediaType, Queue, WrapUpCode
