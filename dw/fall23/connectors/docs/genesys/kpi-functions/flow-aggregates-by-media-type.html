2.9. Flow Aggregates By Media Type (FA_A)¶

2.9.1. FA_A_FlowCount¶

Flow Aggregates By Media Type - Flow Count - actively updated with notifications

This KPI function returns the number of communications that entered a flow.

Parameters: Flow, MediaType, Queue

2.9.2. FA_A_FlowDisconnectCount¶

Flow Aggregates By Media Type - Flow Disconnect Count - actively updated with notifications

This KPI function returns the number of times an entry spent time in a flow until it was disconnected.

Parameters: Flow, MediaType, Queue

2.9.3. FA_A_FlowDisconnectMax¶

Flow Aggregates By Media Type - Flow Disconnect Max - actively updated with notifications

This KPI function returns the longest amount of time that an entry spent in the flow before disconnecting.

Parameters: Flow, MediaType, Queue

2.9.4. FA_A_FlowDisconnectMin¶

Flow Aggregates By Media Type - Flow Disconnect Min - actively updated with notifications

This KPI function returns the shortest amount of time that an entry spent in the flow before disconnecting.

Parameters: Flow, MediaType, Queue

2.9.5. FA_A_FlowDisconnectSum¶

Flow Aggregates By Media Type - Flow Disconnect Sum - actively updated with notifications

This KPI function returns the total amount of time spent in a flow for communications that were abandoned.

Parameters: Flow, MediaType, Queue

2.9.6. FA_A_FlowExitCount¶

Flow Aggregates By Media Type - Exit Count - actively updated with notifications

This KPI function returns the total number of times an entry was in a flow until it exited the flow to another destination, such as an ACD queue, ACD voicemail, or a user.

Parameters: Flow, MediaType, Queue

2.9.7. FA_A_FlowExitMax¶

Flow Aggregates By Media Type - Exit Max - actively updated with notifications

Parameters: Flow, MediaType, Queue

2.9.8. FA_A_FlowExitMin¶

Parameters: Flow, MediaType, Queue

2.9.9. FA_A_FlowExitSum¶

Flow Aggregates By Media Type - Exit Sum - actively updated with notifications

This KPI function returns the total amount of time spent in a flow for communications that exited the flow regularly.

Parameters: Flow, MediaType, Queue
