2.11. Queue Observations (QO_A)¶

2.11.1. QO_A_ActiveUsersCount¶

Queue Observations - Active Users Count - actively updated with notifications

This KPI function returns the observed number of active users in a queue.

Parameters: Queue

2.11.2. QO_A_AvailableAgentsCount¶

Queue Observations - Available Agents Count - actively updated with notifications

This KPI function returns the number of agents that are off queue and in the Available status.

Parameters: Queue

2.11.3. QO_A_AwayAgentsCount¶

Queue Observations - Away Agents Count - actively updated with notifications

This KPI function returns the number of agents that are off queue and in the Away status.

Parameters: Queue

2.11.4. QO_A_BreakAgentsCount¶

Queue Observations - Break Agents Count - actively updated with notifications

This KPI function returns the number of agents that are off queue and in the Break status.

Parameters: Queue

2.11.5. QO_A_BusyAgentsCount¶

Queue Observations - Busy Agents Count - actively updated with notifications

This KPI function returns the number of agents that are off queue and in the Busy status.

Parameters: Queue

2.11.6. QO_A_CommunicatingAgentsCount¶

Queue Observations - Communicating Agents Count - actively updated with notifications

This KPI function returns the number agents currently on a call.

Parameters: Queue

2.11.7. QO_A_IdleAgentsCount¶

Queue Observations - Idle Agents Count - actively updated with notifications

This KPI function returns the number of agents who are on queue and are available to take interactions, but are not working with interactions.

Parameters: Queue

2.11.8. QO_A_InteractingAgentsCount¶

Queue Observations - Interacting Agents Count - actively updated with notifications

This KPI function returns the number of agents working with interactions. This number includes agents completing after-call work.

Parameters: Queue

2.11.9. QO_A_MealAgentsCount¶

Queue Observations - Meal Agents Count - actively updated with notifications

Parameters: Queue

Parameters: Queue

2.11.11. QO_A_MemberAgentsCount¶

Queue Observations - Member Agents Count - actively updated with notifications

This KPI function returns the observed number of agents in a queue.

Parameters: Queue

2.11.12. QO_A_NotRespondingAgentsCount¶

Queue Observations - Not Responding Agents Count - actively updated with notifications

This KPI function returns the number of agents that did not answer an offered interaction. The agent could still be handling and completing other previous interactions.

Parameters: Queue

2.11.13. QO_A_OffQueueAvailableAgentsCount¶

Queue Observations - Off Queue Available Agents Count - actively updated with notifications

This KPI function returns the observed number of available agents that are off queue.

Parameters: Queue

2.11.14. QO_A_OffQueueOfflineAgentsCount¶

Queue Observations - Off Queue Offline Agents Count - actively updated with notifications

This KPI function returns the observed number of offline agents that are off queue.

Parameters: Queue

2.11.15. QO_A_OfflineAgentsCount¶

Queue Observations - Offline Agents Count - actively updated with notifications

Parameters: Queue

2.11.16. QO_A_OnQueueAgentsCount¶

Queue Observations - On Queue Agents Count - actively updated with notifications

This KPI function returns the number of agents that are on queue for a specific queue.

2.11.17. QO_A_OnQueueInteractingAgentsCount¶
