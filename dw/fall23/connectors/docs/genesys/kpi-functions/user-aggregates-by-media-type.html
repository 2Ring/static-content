2.13. User Aggregates By Media Type (UA_A)¶

2.13.1. UA_A_ACWCount¶

User Aggregates By Media Type - ACW Count - actively updated with notifications

This KPI function returns the number of times agents completed after-call work.

Parameters: MediaType, Queue, User

2.13.2. UA_A_ACWSum¶

User Aggregates By Media Type - ACW Sum - actively updated with notifications

This KPI function returns the total time spent doing after-call work (ACW).

Parameters: MediaType, Queue, User

2.13.3. UA_A_AlertCount¶

User Aggregates By Media Type - Alert Count - actively updated with notifications

This KPI function returns the number of times an agent was alerted.

Parameters: MediaType, Queue, User

2.13.4. UA_A_AlertSum¶

User Aggregates By Media Type - Alert Sum - actively updated with notifications

This KPI function returns the total amount of time that the Genesys Cloud alerts agents before the agent accepts or declines an interaction.

Parameters: MediaType, Queue, User

2.13.5. UA_A_ContactingSum¶

User Aggregates By Media Type - Contacting Sum - actively updated with notifications

This KPI function returns the total amount of time that it takes to establish a connection with the station on an outbound call.

Parameters: MediaType, Queue, User

2.13.6. UA_A_DialingSum¶

User Aggregates By Media Type - Dialing Sum - actively updated with notifications

This KPI function returns the total amount of time an agent spent dialing.

Parameters: MediaType, Queue, User

2.13.7. UA_A_HandleCount¶

User Aggregates By Media Type - Handle Count - actively updated with notifications

This KPI function returns the total number of ACD and non-ACD interactions that agents were involved in, including answered interactions, transfers, callbacks, and outbound calls. Handle increments in the interval in which the interaction ends, after the agent completes any after-call work and completes the interaction.

Parameters: MediaType, Queue, User

2.13.8. UA_A_HandleSum¶

User Aggregates By Media Type - Handle Sum - actively updated with notifications

This KPI function returns the total amount of time an agent spent handling an interaction.

Parameters: MediaType, Queue, User

2.13.9. UA_A_HoldCount¶

User Aggregates By Media Type - Hold Count - actively updated with notifications

This KPI function returns the number of interactions in which the interaction went on hold.

Parameters: MediaType, Queue, User

2.13.10. UA_A_HoldSum¶

User Aggregates By Media Type - Hold Sum - actively updated with notifications

This KPI function returns the cumulative hold time for all interactions.

Parameters: MediaType, Queue, User

2.13.11. UA_A_TalkCompleteCount¶

User Aggregates By Media Type - Talk Complete Count - actively updated with notifications

This KPI function returns the number of times an agent was talking/interacting.

Parameters: MediaType, Queue, User

2.13.12. UA_A_TalkSum¶

User Aggregates By Media Type - Talk Sum - actively updated with notifications

This KPI function returns the amount of time spent interacting with a type of media.

Parameters: MediaType, Queue, User

2.13.13. UA_A_TransferredCount¶

User Aggregates By Media Type - Transferred Count - actively updated with notifications

This KPI function returns the total number of interactions transferred. This includes blind and consult transfers.

Parameters: MediaType, Queue, User
