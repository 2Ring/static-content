2.16. User Status Aggregates Active (USA_A)¶

Unlike the User Status Aggregates Passive KPIs, these metrics are only updated when an agent’s state changes.

2.16.1. USA_A_AgentRoutingCommunicatingTimeSum¶

User Status Aggregates - Agent Routing Communicating Time Sum - actively updated with notifications

This KPI function returns the amount of time an agent spent in the Communicating status.

Parameters: Queue, User
