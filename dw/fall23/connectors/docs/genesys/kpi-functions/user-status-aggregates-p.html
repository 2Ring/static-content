2.17.1. USA_P_AgentRoutingCommunicatingTimeSum¶

User Status Aggregates - Agent Routing Communicating Time Sum - passively updated (produces more API requests for your organization)

This KPI function returns the total amount of time an agent spent in the Communicating status.
