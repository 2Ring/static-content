2.2. Agent Statistics¶

This Grid function displays per-agent summary for all agents for the current day.

For more information, see the NICE CXone API documentation.

2.2.1. Columns¶

  • Agent Id - This is a unique, system-generated ID assigned to each Agent.

  • Acw Time - The length of time that the Agent spent in the after call work (acw) state.

  • Agent Offered - The number of Contacts offered to the Agent.

  • Available Time - The length of time that the Agent spent in the available state.

  • Consult Time - The length of time that the Agent spent in consult state.

  • Inbound Avg Talk Time - The length of time that represents the average time per inbound call for the Agent.

  • Inbound Handled - The number of inbound calls handled by the Agent.

  • Inbound Talk Time - The total length of time the Agent spent talking to contacts on inbound calls.

  • Inbound Time - The length of time that the Agent spent on inbound calls.

  • Login Time - The length of time the Agent was logged into the system.

  • Occupancy - The percent time the Agent was in the (talk/acw) work state.

  • Outbound Avg Talk Time - The average time per outbound call the Agent spent talking to contacts.

  • Outbound Handled - The number of outbound calls handled.

  • Outbound Talk Time - The total length of time the Agent spent talking to contacts on outbound calls.

  • Outbound Time - The length of time the Agent spent handling outbound contacts.

  • Percent Refused - The percent of total contacts offered to the Agent, which the Agent never answered.

  • Refused - The total number of calls refused by the Agent.

  • Total Avg Handle Time - The length of time that represents the average time spent handling all calls.

  • Total Avg Talk Time - The average time the Agent spent talking to contacts, including hold time and conference time.

  • Total Handled - The number of calls the Agent handled.

  • Total Talk Time - The total length of time the Agent spent talking to contacts on inbound and outbound calls.

  • Unavailable Time - The length of time that the Agent spent in the unavailable state.

  • Working Rate - The percentage of total login time the Agent was available or actively handling Contacts.

2.2.2. Calculation Parameters¶

The Grid function has these optional parameters:

  • Skills - Allows you to select from a list of skills using a logical OR operation for filtering. When you use this parameter, any object that matches at least one of the selected skills will be included in the results. - List of Skill names.

  • Teams - List of Team names.
