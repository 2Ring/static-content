2.5. Skill Statistics RT¶

This Grid function displays the activity information for all skills.

For more information, see the NICE CXone API documentation.

2.5.1. Columns¶

  • Skill Id - the ID of the Skill.

  • Skill Name - the name of the Skill.

  • Agents Acw - the number of Agents in ACW on Skill.

  • Agents Available - the number of Agents available on Skill.

  • Agents Idle - the number of Agents idle on Skill.

  • Agents Logged In - the number of Agents logged into Skill.

  • Agents Unavailable - the number of Agents unavailable on Skill.

  • Agents Working - the number of Agents working on Skill.

  • Business Unit Id - the ID of Business Unit.

  • Contacts Active - the number of active Contacts for Skill.

  • Earliest Queue Time - the time of the earliest queued contact for Skill.

  • Email From Address - the email address, that is used to send emails from the Skill to customers.

  • Is Active - indicates if the Skill is active.

  • In SLA - the number of contacts the agents handled within the defined service level threshold.

  • Is Natural Calling - indicates if the Skill is in Natural calling mode.

  • Is Outbound - indicates if Skill is outbound.

  • Out SLA - the number of contacts agents handled outside the specified service level threshold.

  • Parked Count - the number of Emails in the parked queue.

  • Personal Queue Count - the number of Emails in the personal queue.

  • Queue Count - the number of Contacts in queue for Skill.

  • Skill Queue Count - the number of Emails in the skill queue.

2.5.2. Calculation Parameters¶

The Grid function has these optional parameters:

  • Skills - Allows you to select from a list of skills using a logical OR operation for filtering. When you use this parameter, any object that matches at least one of the selected skills will be included in the results. - list of Skill names
