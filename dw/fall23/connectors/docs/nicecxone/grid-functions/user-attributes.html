2.6. User Attributes¶

This Grid function displays static data for agents.

For more information, see the NICE CXone API documentation.

2.6.1. Columns¶

  • Agent Id - the ID of the Agent.

  • Agent Name - the name of the agent.

  • Agent First Name - the first name of the Agent.

  • Agent Last Name - the last name of the Agent.

  • Agent Middle Name - the middle name of the Agent.

  • User Id - the User ID for the Agent.

  • User Name - the username for the Agent.

  • User Type - the type of user for WFM.

  • Assigned Skills - the names of Skills for the Agent.

  • Assigned Team - the name of the team for the Agent.

  • Assigned Team Id - the ID of the team for the Agent.

  • Email - the email address of the Agent.

  • Telephone 1 - the optional telephone number for Agent.

  • Telephone 2 - the optional telephone number for Agent.

2.6.2. Calculation Parameters¶

The Grid function has these optional parameters:

  • Agents - List of Agent names.

  • Skills - Allows you to select from a list of skills using a logical OR operation for filtering. When you use this parameter, any object that matches at least one of the selected skills will be included in the results. - List of Skill names.

  • Teams - List of Team names.
