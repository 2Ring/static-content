1.1. Agent State¶

The Agent State KPI functions display aggregated current state of all agents.

For more information, see the NICE CXone API documentation.

Depending on the KPI function, the following parameters may be used:

  • AgentStates - List of agent states

  • Skills - Allows you to select from a list of skills using a logical OR operation for filtering. When you use this parameter, any object that matches at least one of the selected skills will be included in the results. - List of Skill names

  • Teams - List of Team names

1.1.1. Agent State - In State¶

This KPI function returns the number of agents that are currently in the specified states.

Parameters: AgentStates, Skills, Teams
