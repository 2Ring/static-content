1.2. Agent Statistics¶

The Agent Statistics KPI functions display aggregated summary for all agents for the current day.

For more information, see the NICE CXone API documentation.

Depending on the KPI function, the following parameters may be used:

  • Skills - Allows you to select from a list of skills using a logical OR operation for filtering. When you use this parameter, any object that matches at least one of the selected skills will be included in the results. - List of Skill names

  • Teams - List of Team names

1.2.1. Agent Statistics - Acw Time¶

This KPI function returns the length of time that the agents spent in the after call work (acw) state.

Parameters: Skills, Teams

1.2.2. Agent Statistics - Agent Offered¶

This KPI function returns the number of contacts offered to the agents.

Parameters: Skills, Teams

1.2.3. Agent Statistics - Available Time¶

This KPI function returns the length of time that the agents spent in the available state.

Parameters: Skills, Teams

1.2.4. Agent Statistics - Consult Time¶

This KPI function returns the length of time that the agents spent in consult state.

Parameters: Skills, Teams

1.2.5. Agent Statistics - Inbound Handled¶

This KPI function returns the number of inbound calls handled by the agents.

Parameters: Skills, Teams

1.2.6. Agent Statistics - Inbound Talk Time¶

This KPI function returns the length of time the agents spent talking to contacts on inbound calls.

Parameters: Skills, Teams

1.2.7. Agent Statistics - Inbound Time¶

This KPI function returns the length of time that the agents spent on inbound calls.

Parameters: Skills, Teams

1.2.8. Agent Statistics - Login Time¶

This KPI function returns the length of time the agents was logged into the system.

Parameters: Skills, Teams

1.2.9. Agent Statistics - Outbound Handled¶

This KPI function returns the number of outbound calls handled.

Parameters: Skills, Teams

1.2.10. Agent Statistics - Outbound Talk Time¶

This KPI function returns the length of time the agents spent talking to contacts on outbound calls.

Parameters: Skills, Teams

1.2.11. Agent Statistics - Outbound Time¶

This KPI function returns the length of time the agents spent handling outbound contacts.

Parameters: Skills, Teams

1.2.12. Agent Statistics - Refused¶

This KPI function returns the number of calls refused by the agents.

Parameters: Skills, Teams

1.2.13. Agent Statistics - Total Handled¶

This KPI function returns the number of calls the agents handled.

Parameters: Skills, Teams

1.2.14. Agent Statistics - Total Talk Time¶

This KPI function returns the length of time the agents spent talking to contacts on inbound and outbound calls.

Parameters: Skills, Teams

1.2.15. Agent Statistics - Unavailable Time¶

This KPI function returns the length of time that the agents spent in the unavailable state.

Parameters: Skills, Teams
