1.3. Skill Statistics¶

The Skill Statistics KPI functions display aggregated reporting summary data for every Skill for the current day.

For more information, see the NICE CXone API documentation.

Depending on the KPI function, the following parameters may be used:

  • Skills - Allows you to select from a list of skills using a logical OR operation for filtering. When you use this parameter, any object that matches at least one of the selected skills will be included in the results. - list of Skill names

1.3.1. Skill Statistics - Abandon Count¶

This KPI function returns the number of abandoned contacts.

Parameters: Skills

1.3.2. Skill Statistics - Connects¶

This KPI function returns the number of times the Personal Connection dialer results in an answer event.

Parameters: Skills

1.3.3. Skill Statistics - Contacts Active¶

This KPI function returns the number of contacts that are currently active in the system.

Parameters: Skills

1.3.4. Skill Statistics - Contacts Handled¶

This KPI function returns the number of contacts handled.

Parameters: Skills

1.3.5. Skill Statistics - Contacts Offered¶

This KPI function returns the number of contacts offered to agents.

Parameters: Skills

1.3.6. Skill Statistics - Contacts Queued¶

This KPI function returns the number of inbound contacts placed in queue. This includes contacts that spent time in the queue and those that were immediately routed to an available agent. This number includes contacts transferred to a skill or an agent. This number does not include transfers followed by a consult.

Parameters: Skills

1.3.7. Skill Statistics - Contacts Out Of SLA¶

This KPI function returns the number of contacts who entered the skills and were handled outside the required SLA service level threshold. This statistic includes abandons based on whether the skill is configured to include them.

Parameters: Skills

1.3.8. Skill Statistics - Contacts Within SLA¶

This KPI function returns the number of contacts who entered the skills and were handled within the required SLA service level threshold. This statistic includes abandons based on whether the skill is configured to include them.

Parameters: Skills

1.3.9. Skill Statistics - Dials¶

This KPI function returns the number of times the Personal Connection dialer initiates a call for a contact.

Parameters: Skills

1.3.10. Skill Statistics - Hold Time¶

This KPI function returns the amount of time that skills were in the hold state.

Parameters: Skills

1.3.11. Skill Statistics - Longest Queue Dur¶

This KPI function returns the amount of time for the one contact which spent the most time in the queue waiting for an agent.

Parameters: Skills

1.3.12. Skill Statistics - Right Party Connects¶

This KPI function returns the number of INTENDED contacts that were reached. Intended contacts are counted as Right Party Connects irrespective of whether there is a positive or negative outcome from the interaction.

Parameters: Skills

1.3.13. Skill Statistics - Total Contact Time¶

This KPI function returns the amount of time that the contact spent in the NICE inContact system.

Parameters: Skills
