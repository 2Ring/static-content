Task States¶

This Grid function calculates task states.

All data for this function is collected using ServiceNow Aggregate API. In case the Aggregate API is disabled, ServiceNow Table API is used.

Columns¶

  • Table Name – the name of the task table.

  • Active – the number of currently active tasks.

  • State – the number of tasks, currently in state defined in the parameter.

Calculation Parameters¶

  • Assignment Group – list of assignment group names.

  • Company Name - list of company names.

  • Priority - list of task priorities.

  • State - list of task states.

  • Table - list of task tables.

Column-mapped Parameters¶

  • Table Name - Table calculation parameter value
