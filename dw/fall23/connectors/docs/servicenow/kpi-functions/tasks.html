Tasks¶

Depending on the KPI function, the following parameters may be used:

  • Assignment Group – list of assignment group names.

  • Company Name – list of company names.

  • Priority – list of numbers of priorities.

  • State – list of numbers of states.

  • Table - list of task tables.

  • Time Period – time period for calculating data. Default setting is Today. Available values are: “LastXDays”, “Period Time”, “Today”.

  • Time Period Count – number of time periods that will be calculated (applies to Time Period parameter: Period Time).

  • Time Period Field – name of a time field used to query data against time period parameter. Available values are: “Closed Time”, “Created Time”, “Opened Time”.

  • Time Period Interval – specifies the size of a time period (applies to Time Period parameter: Period Time). Available values are: “Week”, “Month” and “Year”.

  • Time Period Offset – offset for a time period (applies to Time Period parameter: Period Time).

    • If it is set to 0, it starts to calculate actual week, month or year.

    • If it set to 1 or more, it starts to calculate from previous X week(s), month(s) or year(s) back.

  • Time Period Shift – number of last X days (applies to Time Period parameter: Last X Days).

Tasks – Active¶

This KPI function returns the number of currently active tasks.

Parameters: Assignment Group, Company Name, Priority, State, Table

Tasks – Closed¶

This KPI function returns the number of tasks, that were closed today.

Parameters: Assignment Group, Company Name, Priority, State, Table

Tasks – Created¶

This KPI function returns the number of tasks, that were created today.

Parameters: Assignment Group, Company Name, Priority, State, Table

Tasks – In Interval¶

This KPI function returns the number of tasks. It supports a group of time period parameters.

Parameters: Assignment Group, Company Name, Priority, Table, Time Period, Time Period Count, Time Period Field, Time Period Interval, Time Period Offset, Time Period Shift

Tasks – Longest Waiting¶

This KPI function returns the queue time of the longest waiting task.

Parameters: Assignment Group, Company Name, Priority, State, Table

Tasks – Opened¶

This KPI function returns the number of tasks, that were opened today.

Parameters: Assignment Group, Company Name, Priority, State, Table

Tasks - State¶

This KPI function returns the number of tasks in states.

Parameters: Assignment Group, Company Name, Priority, State, Table
