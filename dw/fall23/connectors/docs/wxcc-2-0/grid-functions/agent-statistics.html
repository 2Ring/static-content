2.2. Agent Statistics¶

The Agent Statistics Grid Function displays aggregated Agent statistics for the current day in a tabular format.

For more information, see the Webex API documentation.

2.2.1. Columns¶

  • Agent Id - the Id of the agent

  • Channel Type - the name of the Channel Type

  • Team Id - the Id of agent’s current team

  • Team Name - the name of agent’s current team

  • Tasks Accepted - the number of assigned tasks that were accepted by the agent

  • Tasks Assigned - the number of tasks assigned to the agent during the agent session

  • Tasks Offered - the number of tasks offered to the agent during the agent session

  • Tasks Rejected - the number of assigned tasks that were rejected by the agent

  • Tasks Transferred - the number of tasks the agent transferred to another agent

  • Total Available Time - the amount of time outside of the idle time when an agent was available and handling contacts after logging in

  • Total Engaged Duration - the amount of time the agent was engaged with customers

  • Total Hold Duration - the amount of time for which customers were put on hold

  • Total Unavailable Time - the amount of time the agent was unavailable

  • Total Wrap Up Duration - the amount of time spent by the agent wrapping up customer interactions

2.2.2. Calculation Parameters¶

The Grid function has these optional parameters:

  • Agents - list of Agent names

  • CompetencyLevel - Minimum profiency level of skill in skill profile, for skill type PROFIENCY

  • Teams - list of Team names

  • Channel Type - list of Channel Types

  • Queues - List of Queue names

  • Skills - List of Skill names

  • SkillsBooleanValue - Boolean value of skill in skill profile, for skill type BOOLEAN

  • SkillsEnumValues - List of values of skill in skill profile, for skill type ENUM

  • SkillsTextValues - List of values of skill in skill profile, for skill type TEXT

If the Skill parameter is filled, it is necessary to fill in some of the parameters SkillsBooleanValue, SkillsEnumValues, SkillsTextValues and CompetencyLevel according to the type of skills specified in the parameter.
The agent will be displayed if assigned a skill profile that includes any of the skills specified in the parameter, combined with the skill value according to the skill type.

2.2.3. Column-mapped Parameters¶

  • Agent Id - Agents calculation parameter value

  • Channel Type - Channel Type calculation parameter value

  • Team Id - Teams calculation parameter value

  • Team Name - Teams calculation parameter value
