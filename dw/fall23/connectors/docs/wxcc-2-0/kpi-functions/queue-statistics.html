1.5. Queue Statistics¶

The Queue Statistics KPI Functions display aggregated Queue statistics for the current day.

For more information, see the Webex API documentation.

Depending on the KPI Function, the following parameters may be used:

  • Queues - list of Queue names

  • Channel Type - list of Channel Types

  • WrapupCodes - list of Wrapup Codes

1.5.1. Queue Statistics - Longest Accepted Queue Time¶

This KPI Function returns the longest time a task spent in the queue before being accepted by an agent.

Parameters: Queues, Channel Type, WrapupCodes

1.5.2. Queue Statistics - Longest Accepted Ringing Time¶
