1.7. Tasks¶

The Tasks KPI Functions calculate real time statistics for Tasks.

Depending on the KPI Function, the following parameters may be used:

  • Queues - list of Queue names

  • Channel Type - list of Channel Types

  • Termination Type - list of values which indicates how the call was terminated

  • Threshold - time in seconds

1.7.1. Tasks - Answered In Service Level¶

This KPI Function returns the number of assigned tasks that were accepted by agents in a certain time.

Parameters: Queues, Channel Type, Threshold

These KPI Functions utilize REST API for initialization and Webhook Notifications for updates.

IVR time is included in the calculation.

1.7.2. Tasks - Ended In Service Level¶

This KPI Function returns the number of tasks that were ended in a certain time.

Parameters - Queues, Threshold, Termination Type
