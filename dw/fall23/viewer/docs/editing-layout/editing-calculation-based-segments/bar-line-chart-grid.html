Bar/Line Chart - Grid¶

A Bar/Line Chart - Grid can display values of one or more calculations. This way, development of multiple calculations can be compared with each other.

For general information about this segment, see Bar/Line Chart - Grid chapter.

  • External Id

    An unique id, that is used to access this Word Cloud via the API. Using this id, the Word Cloud can be used by external applications, such as 2Ring Gadgets. The External Id has to be unique across the entire Business Unit.

  • Calculation

    The calculation options control, how the displayed values are calculated.
    For calculation options, see Editing Table-Valued Calculations chapter.

    • Use Shared Calculation

      Whether shared calculation should be used for generating results.

    • Shared Calculation

      Available only if “Use Shared Calculation” is enabled
      A shared calculation to be used.

  • Appearance

    The style of the Timeline Chart

    • Value Axis Orientation

      Orientation of the value axis.
      Available options are: Horizontal and Vertical

      Horizontal value axis orientation supports only Bar type calculations. Calculations with any other type selected are hidden in the grid. Also, rotation of axis labels with not enough space is not possible when the horizontal value axis orientation is set.

    • Show Legend

      When enabled, the chart legend is displayed.

      • Position

        Required
        Position of legend.
        Available options are Left, Right, Top and Bottom

      • Text Color

        Inheritable
        For more information see Color Field section.
        Color of the text in legend.

      • Background Color

        Inheritable
        For more information see Color Field section.
        Color of the background in legend.

      • Text Style

        Inheritable
        For more information see Text Style section.
        Type of font to use and whether bold or italic text should be used.

      • Font Size

        Inheritable
        Size of the font. Measured in permilles of the entire Layout’s height.

      • Font Size Coefficient

        Inheritable
        The font size is calculated by multiplying available height for the text by font size coefficient.

    • Label Axis

      Configuration of horizontal chart axis

      • Column

        Required
        A column that represents the label axis.

      • Sort Column

        The column, data should be ordered by.

      • Sorting Direction

        Whether the column should be ordered in ascending or descending order.

      • Display Format

        Format in which value should be displayed. When Raw is selected, the value is displayed without any transformation.

      • Custom Display Format

        Only if Custom is selected as a Display Format
        For more information see Custom Display Format section.
        Custom format in which value should be displayed.

      • Auto Rotate Labels

        Whether axis labels should be automatically rotated when there is not enough space for them.

      • Text Style

        Inheritable
        For more information see Text Style section.
        Labels do not support Custom CSS in the Text Style. If Theme Text Style, with Custom CSS is chosen, Custom CSS will not be applied
        Type of font to use and whether bold or italic text should be used.

      • Font Size

        Inheritable
        Size of the font. Measured in permilles of the entire Layout’s height.

      • Font Size Coefficient

        Inheritable
        The font size is calculated by multiplying available height for the text by font size coefficient.

      • Show Grid Lines

        Whether the lines, separating the items of the axis should be displayed.

    • Plot Area

      Configuration of the chart’s plot area.

      • Margin

        Inheritable
        Controls the height of the empty space above the plot area. Is measured in permilles of the whole Layout’s height.

      • Background Color

        Inheritable
        For more information see Color Field section.
        Color of the plot area.

    • Colors

      For ease of access, all aforementioned color related options are once again located in the Colors group.

      • Text Color

        Inheritable
        For more information see Color Field section.
        Color of the text. This color is applied to both axes labels and text in legend.

      • Background Color

        Inheritable
        For more information see Color Field section.
        Color of the background.

      • Plot Area Background Color

        Inheritable
        For more information see Color Field section.
        Background color of the plot area.

      • Axes Color

        Inheritable
        For more information see Color Field section.
        Color of the both axes lines.

      • Grid Lines Color

        Inheritable
        For more information see Color Field section.
        Color of the horizontal and vertical grid lines.

  • Value Axes

    Manually orderable
    List of Chart’s value axes. For more information see Value Axes chapter.

  • Series

    Manually orderable
    List of Chart’s series. For more information see Series chapter.
