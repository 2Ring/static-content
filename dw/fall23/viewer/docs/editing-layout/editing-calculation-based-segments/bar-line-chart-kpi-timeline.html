Bar/Line Chart - Kpi Timeline¶

A Bar/Line Chart - Kpi Timeline can display values of one or more calculations, over the course of a specific period. The period is represented by an horizontal axis which spans the whole period.

For general information about this segment, see Bar/Line Chart - KPI Timeline chapter.

  • Period Settings

    This group defines period over which we want to display values.

    • Period

      Required
      A desired type of time period to be displayed in chart.
      Available options are Day, Week, Month, Year and Sliding Interval

    • Number of Displayed Periods

      Required
      The number of days, weeks, months or years to display.

      When Number of Displayed Periods is more than 1, each period is represented by its own chart. Charts are stacked vertically inside of the segment.

    • Displayed Period/Newest Displayed Period

      Required
      When multiple periods should be displayed this option specifies which will be the newest displayed period.
      When only one period should be displayed this option defines which period it will be.
      Available options are:

      • This Period

        The calendar displays the currently ongoing period.

      • Last Period

        The calendar displays the previous period.

      • Other Period

        If this option is chosen, how far into the past the Displayed Period/Newest Displayed Period is has to be provided.

    • Granularity

      Required
      Available only when “Day”, “Year” or “Sliding Interval” period is selected. When “Month” or “Week” period is selected day granularity is used.
      The time interval each series item represent.

    • Show Current Day/Hour/…

      Available only when “This Period” is selected as Displayed Period/Newest Displayed Period.
      Whether the value for ongoing series item (interval) should be displayed. Value is being automatically updated if displayed.

    • Number of Periods

      Available only if “Sliding Interval” is selected as a Period
      Number of periods to be displayed.

    • Number of intervals in Period

      Available only if “Sliding Interval” is selected as a Period
      Number of selected intervals to be displayed within each period.

    • Filter Hours

      Available only when “Day” period is selected or when “Sliding Interval” with granularity less than a “Day” is selected.
      When Day, or Sliding Interval with granularity less than a Day is selected as a period, time-based intervals are shown as series items. This options is used to limit which items should be shown.

      • From

        Required
        Items are shown only if their associated intervals started after or at this time.

      • To

        Required
        Items are shown only if their associated intervals ended before or at this time.

    • Time zone

      Inheritable
      Specifies based on which time zone values should be calculated.

    • Weekdays

      When “Day” period is selected, if non-working day should be shown, nearest previous working day is displayed instead.
      When “Day” granularity is selected, non-working days are not displayed in the chart.
      The Timeline Chart can be configured to display only certain relevant days. Has effect only if Day granularity or period is selected.

      • First Day Of Week

        Inheritable
        Available only when “Week”, “Month” or “Year” period is selected.
        Specifies, which day should be displayed and considered as first of each week.

      • Monday, …, Sunday

        Inheritable
        Available options are *Working Day and “Non-Working Day”*
        Visibility of each day can be configured with its corresponding dropdown. Non-Working days are hidden.

  • Appearance

    The style of the Bar/Line Chart - Kpi Timeline.

    • Show Headers

      When enabled, headers describing shown periods are displayed

      • Height

        Inheritable
        Height of the header, that is measured in permilles of the entire Layout’s height.

      • Text Color

        Inheritable
        For more information see Color Field section.
        Color of the text.

      • Background Color

        Inheritable
        For more information see Color Field section.
        Color of the background.

      • Text Style

        Inheritable
        For more information see Text Style section.
        Type of font to use and whether bold or italic text should be used.

      • Font Size Coefficient

        Inheritable
        The font size is calculated by multiplying available height for the text by font size coefficient.

    • Show Legend

      When enabled, the chart legend is displayed.

      • Position

        Required
        Available options are “Left”, “Right”, “Top” and “Bottom”
        Position of legend.

      • Text Color

        Inheritable
        For more information see Color Field section.
        Color of the text.

      • Background Color

        Inheritable
        For more information see Color Field section.
        The background color of the legend.

      • Text Style

        Inheritable
        For more information see Text Style section.
        Type of font to use and whether bold or italic text should be used.

      • Font Size

        Inheritable
        Size of the font. Measured in permilles of the entire Layout’s height.

      • Font Size Coefficient

        Inheritable
        The font size is calculated by multiplying available height for the text by font size coefficient.

    • Date Axis

      Configuration of horizontal chart axis

      • Auto Rotate Labels

        Whether axis labels should be automatically rotated when there is not enough space for them.

      • Text Style

        Inheritable
        For more information see Text Style section.
        Type of font to use and whether bold or italic text should be used.

      • Font Size

        Inheritable
        Size of the font. Measured in permilles of the entire Layout’s height.

      • Font Size Coefficient

        Inheritable
        The font size is calculated by multiplying available height for the text by font size coefficient.

      • Show Grid Lines

        Whether the lines, separating the items of the axis should be displayed.

    • Plot Area

      Configuration of the chart’s plot area.

      • Margin

        Inheritable
        Controls the height of the empty space above the plot area. Is measured in permilles of the whole Layout’s height.

      • Background Color

        Inheritable
        For more information see Color Field section.
        Color of the plot area.

        Color of the text. This color is applied to both axes labels and text in legend.

      • Background Color

        Inheritable
        For more information see Color Field section.
        Color of the background.

      • Header Text Color

        Inheritable
        For more information see Color Field section.
        Color of the header text.

      • Header Background Color

        Inheritable
        For more information see Color Field section.
        Color of the header background.

      • Plot Area Background Color

        Inheritable
        For more information see Color Field section.
        Background color of the plot area.

      • Axes Color

        Inheritable
        For more information see Color Field section.
        Color of the both axes lines.

      • Grid Lines Color

        Inheritable
        For more information see Color Field section.
        Color of the horizontal and vertical grid lines.
