Value Axes¶

The axis determines how the values of the series are displayed.

The Value Axes list is orderable. When multiple axes are displayed and positioned on the same side of the chart, their order defines in which order will they be shown. The higher the axis is in the list the closer to the chart it will be displayed.

  • Title

    Required
    Title of the Value Axis used as its label in chart and also in list of Value Axes.

  • Show Axis

    Whether the axis should be displayed in chart.

    • Position

      Required
      Position of the axis within the plot area.
      Available options are Left and Right

    • Show Grid Lines

      Whether the lines, separating the labels on the axis should be displayed.

    • Show Title

      Whether axis title should be displayed.

      • Text Style

        Inheritable
        For more information see Text Style section.
        Title does not support Custom CSS in the Text Style. If Theme Text Style, with Custom CSS is chosen, Custom CSS will not be applied
        Type of font to use and whether bold or italic text should be used.

      • Font Size

        Inheritable
        Size of the font. Measured in permilles of the entire Layout’s height.

      • Font Size Coefficient

        Inheritable
        The font size is calculated by multiplying available height for the text by font size coefficient.

    • Show Labels

      Required
      When enabled, axis labels are displayed along the axis in regular intervals.

      • Display Format

        Required
        Label’s format. When Raw is selected, the labels are displayed without any transformation.

      • Custom Display Format

        Available only when “Custom” is selected as a Display Format
        For more information see Custom Display Format section.
        Custom format of the label.

      • Text Style

        Inheritable
        For more information see Text Style section.
        Labels do not support Custom CSS in the Text Style. If Theme Text Style, with Custom CSS is chosen, Custom CSS will not be applied
        Type of font to use and whether bold or italic text should be used.

      • Font Size

        Inheritable
        Size of the font. Measured in permilles of the entire Layout’s height.

      • Font Size Coefficient

        Inheritable
        The font size is calculated by multiplying available height for the text by font size coefficient.

  • Choose Range Manually

    When disabled, the axis range is automatically adjusted, based on displayed values.
    When enabled, the axis range has to be manually set.

    • Min

      Minimum of the range.

    • Max

      Maximum of the range.

  • Thresholds

    Thresholds are graphically represented by customizable horizontal line across whole chart area. For more information see Thresholds chapter.
