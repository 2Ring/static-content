Source Tables¶

  • Function

    Required
    Connector provided calculation function, used for calculation of the results.

  • Alias

    Required
    Source Table appears under this alias in dropdowns and lists.

    Changes made to the Alias field are not applied to Filters and Join Condition fields automatically. Alias in these fields needs to be manually changed.

  • Join Type

    Required
    Not available for the first Source Table in the list.
    A join type, that is used to join this Source Table with other tables in the Grid.
    Available options are INNER, LEFT, RIGHT, FULL and CROSS

  • Join Condition

    Required
    Not available for the first Source Table in the list.
    For more information see Expression builder chapter.
    An expression that determines how this Source Table is joined with other tables in the Grid.

    Order of Source Tables in Grid affects list of columns available to be used in Join Condition. Source Table can be joined only with other Source Tables that are defined before it in list of Source Tables.

    Custom columns can not be used in the join condition.

  • Parameters

    For more information refer to the Parameters chapter.

  • Is Historical

    Whether source table should provide historical values.

    • Interval Type

      Required
      Specifies the type of interval displayed value should be from.

    • Which Interval

      Required
      Lets an user choose from which interval, of selected type, data should be displayed. For each interval type current, previous or other interval can be chosen. If other option is chosen, how far into the past the displayed interval is has to be provided.

    • Number of intervals

      Required
      Specify the exact number of selected interval types to be displayed.

  • Advanced

    Advanced Source Table settings.

    • Time Zone

      Inheritable
      Specify based on which time zone values should be calculated.

    • Refresh Interval

      Inheritable
      Interval in seconds, between each refresh of the calculation’s data.

    • Log Retention

      Inheritable
      Result of all calculations are logged, for configured number of days. Unsuccessful calculations are also logged. These logs can help with troubleshooting.

    • Calculation Id

      Serves for information purposes only
      Calculation Id is available only when calculation-based function is selected
      Id of the calculation. Calculation Id is not being automatically updated after changing calculation. Manual refresh is required to show up-to-date value.
