Editing Table-Valued Calculations¶

The calculation options control, how its results are calculated.

  • Source Tables

    Manually orderable
    The list of data tables which provide displayed data. For more information see Source Tables chapter.

  • Custom Columns

    List of custom columns. For more information see Custom Columns chapter.

  • Sort Columns

    Manually orderable
    List of Grid’s sort columns. For more information see grid/sort-columns chapter.

  • Filters

    Filtering of displayed values is configured in this group.

    • Maximum Number Of Rows

      Maximum number of rows to be displayed.

      Rows filtered out by this setting are counted in total values.

    • Filter

      For more information see Expression builder chapter.
      The values are filtered using the configured expression. The configured filter is always applied.

    • Dynamic Filter

      For more information see Dynamic Filter chapter.
      The values are filtered using the configured expression containing value placeholders. Values of these placeholders are received in the URL.

      Rows filtered out by this setting are counted in total values.
