Row Conditional Styles¶

Row Conditional Style is used to alter colors in specific rows. This way row can be “highlighted”.

The Row Conditional Style list is orderable. When condition of multiple Row Conditional Styles is met for the same row, first in line is applied.

  • Column

    The column whose value triggers the row conditional style.

  • Mode

    For information about setting the condition, see Condition section.
    A condition that needs to be met for the row conditional style to apply.

  • Description

    A short definition of the usage of the row conditional style.

  • Appearance

    A style of the condition.

    • Text Color

      For more information see Color Field section.
      Color of the text.

    • Alternate Row Text Color

      For more information see Color Field section.
      Alternating color of the text.

    • Background Color

      For more information see Color Field section.
      Color of the background.

    • Alternate Row Background Color

      For more information see Color Field section.
      Alternating color of the background.

    • Text Style

      Inheritable
      For more information see Text Style section.
      Type of font to use and whether bold or italic text should be used.

    • Borders

      Inheritable
      For more information see Borders Style section.
      The style, width and color settings for the row conditional style borders.

    • Blink Background

      Whether a background should blink, when the condition is met.

      • Color

        For more information see Color Field section.
        Color of the blinking background.

  • Play Audible Alert

    Whether audio sound should be played, when the condition is met.

    • Repetition

      Defines how many times audio alert should be played.

      • Repeat

        The sound is continuously repeated while the bad value is present.

      • Don’t Repeat

        The sound is played only once, when the bad value is first displayed.

    • Audio File

      The alert sound that will be played.
