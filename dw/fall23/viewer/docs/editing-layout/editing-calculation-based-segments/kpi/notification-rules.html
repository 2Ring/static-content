Notification Rules¶

  • Name

    Name of the notification rule.

  • Condition

    The condition which is used to determine when the notification should be sent.

    • Operator

      Required
      Operator used to compare the values. Value displayed in the KPI is always on the left side of the operator. The configured compare value is on the right side.
      Available options are >, <, ≥, ≤, ≠ and =

    • Value

      Required
      Value with which calculated value should be compared.

  • Delay Alerts

    If enabled, notifications are sent only if they are met for certain amount of time.

    • Delay

      Required
      Defines a time period during which the condition has to be continuously met for notification to be sent.

  • Suppress Consecutive Alerts

    To avoid too frequent alerts consecutive alerts can be suppressed. Only alerts generated by associated notification rule are suppressed.

    • Time Period

      Required
      A time period during which the consecutive alerts would be suppressed.

  • Messages

    List of messages that will be sent when notifying. For more information see Messages chapter.
