  • Name

    A name of the notification message.

  • Channel

    Required
    For more information about notification channels see ../../../../../administration-tool-guide/business-units/notification-channels chapter.
    A notification channel that will be used to sent notification.

  • Template

    Required
    For more information about templates see ../../../../../administration-tool-guide/business-units/notification-channels/message-templates chapter.
    Defines content/text of the notification. One of pre-defined templates or custom template can be selected.

  • Custom Subject

    For more information see Expression builder chapter.
    A subject of the notification message. Subject is often used in channel notification popups instead of whole text.
    This option can not be edited when pre-defined message template has been selected.
    Placeholders can be used in message subject which will be replaced with corresponding value or text at the time of the notification. List of all placeholders is available here.

  • Custom Message Body

    Required
    For more information about different types of notification channels and their supported formats see ../../../../../administration-tool-guide/notification-settings/notification-channel-types chapter.
    For more information see Expression builder chapter.
    A text of the notification message. It has to be in correct format supported by each notification channel.
    This option can not be edited when pre-defined message template has been selected.
    Placeholders can be used in message subject which will be replaced with corresponding value or text at the time of the notification. List of all placeholders is available here.
    Format of a body is specific to each notification channel type and is documented in corresponding subchapter in ../../../../../administration-tool-guide/notification-settings/notification-channel-types chapter.

  • Recipients

    List of recipient to whom the message will be sent. For more information see Recipients chapter.
