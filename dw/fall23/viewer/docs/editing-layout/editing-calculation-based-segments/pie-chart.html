Pie Chart¶

The Pie Chart is best used, when comparing the results of multiple different calculations. Each calculation is presented as a segment of a pie shaped chart. The chart can be accompanied with a legend, displaying additional information about each segment.

For general information about this segment, see Pie Chart chapter.

  • Appearance

    The style of a Pie Chart

    • Type

      Required
      Type of the Pie Chart.
      Available options are:

      • Doughnut

        The chart is shaped like a doughnut, with a hole in the middle.

      • Pie

        The chart is represented by a fully filled circle.

    • Cut Out

      Whether chart’s segments are separated from each other by a small gap.

    • Show Segment Labels

      When enabled, labels are visible on top of each segment.

      • Display When

        Required
        Whether the labels should be visible, when its segment isn’t big enough to contain it.
        Available options are Always and If Enough Space

      • Mode

        Required
        Defines what type of information is shown in labels.
        Available options are Title, Value and Percentage

      • Display Format

        Required
        Available only when “Value” or “Percentage” is selected as Mode.
        Format of segment label.

      • Custom Display Format

        Available only when “Custom” is selected as a Display Format
        For more information see Custom Display Format section.
        Custom format of segment label.

      • Arc Text

        When enabled, the label is curved along the segment.

      • Position

        Required
        The relative distance of the label from the middle of chart. 0% means closest to the middle and 100% means furthest.

        Position value can go even above 100% resulting in label to be shown outside of the segment. But in such scenario full visibility of labels is not guaranteed.

      • Label Color

        Inheritable
        For more information see Color Field section.
        Color of the label.

      • Text Style

        Inheritable
        For more information see Text Style section.
        Type of font to use and whether bold or italic text should be used.

      • Font Size

        Inheritable
        Size of the font. Measured in permilles of the entire Layout’s height.

      • Font Size Coefficient

        Inheritable
        The font size is calculated by multiplying available height for the text by font size coefficient.

    • Show Legend

      When enabled, the legend is displayed. The legend displayed additional information about each segment.

      • Show Values

        Whether the actual value, that the segment represents, should be displayed in legend.

        • Display Format

          Required
          Format of the actual value.

        • Custom Display Format

          Available only when “Custom” is selected as a Display Format
          For more information see Custom Display Format section.
          Custom format of the actual value.

      • Show Percentages

        Whether the percentage, that each segment takes up from the whole chart, should be displayed in legend.

        • Display Format

          Required
          Format of the percentage.

        • Custom Display Format

          Available only when “Custom” is selected as a Display Format
          For more information see Custom Display Format section.
          Custom format of the actual value.

      • Legend Content

        Content of the legend.
        Available options are Segment Title and Segment Interval

      • Position

        Required
        Position of legend.
        Available options are Left, Right, Top and Bottom

      • Text Color

        Inheritable
        For more information see Color Field section.
        Color of the text.

      • Background Color

        Inheritable
        For more information see Color field section.
        Color of the legend’s background.

      • Text Style

        Inheritable
        For more information see Text Style section.
        Type of font to use and whether bold or italic text should be used.

      • Font Size

        Inheritable
        Size of the font. Measured in permilles of the entire Layout’s height.

      • Font Size Coefficient

        Inheritable
        The font size is calculated by multiplying available height for the text by font size coefficient.

      • Custom Null Value

        Inheritable
        A custom value that is displayed when the value is not available.

    • Colors

      For ease of access, all aforementioned color related options are once again located in the Colors group.

      • Background Color

        Inheritable
        For more information see Color Field section.
        Color of the background.

      • Segment Label Color

        Inheritable
        For more information see Color Field section.
        Color of the segment label.

      • Legend Text Color

        Inheritable
        For more information see Color Field section.
        Color of the legend text.
