Segment¶

Pie Chart segment represent a calculation. Each segment is represented in the chart by a corresponding portion of the whole circle.

  • External Id

    Only available if Shared Calculation is not used
    An unique id, that is used to access this Segment via the API. Using this id, the Segment can be used by external applications, such as 2Ring Gadgets. The External Id has to be unique across the entire Business Unit.

  • Title

    The title of the segment.

  • Calculation

    For calculation options, see Editing Scalar-Valued Calculations chapter.

    • Use Shared Calculation

      Whether shared calculation should be used for generating results.

    • Shared Calculation

      Available only if “Use Shared Calculation” is enabled
      A shared calculation to be used.

    • Column

      Available only if Grid Shared Calculation is used and Shared Calculation is selected
      Specifies a column from which the value should be displayed.

    • Row Filter

      Available only if Grid Shared Calculation is used and Shared Calculation is selected
      Row Filter is used to determine a specific row from the Grid Shared Calculation that should be used. For more information see Row Filters chapter.

  • Use Test Value

    Test value is used to force the pie chart segment to display a user defined value.

    • Value

      A value to be displayed.

  • Appearance

    The style of a Segment

    • Background Color

      For more information see Color field section.
      Background color of the pie chart segment.

    • Label Color

      Inheritable
      For more information see Color field section.
      Color of the pie chart segment’s label.

    • Show In Legend

      Whether the segment should be displayed in the legend.
