Word Cloud¶

Word Cloud graphically displays words in a size that depends on the weight of the word. Size of the words in a source is directly proportional to weight.

For general information about this segment, see Word Cloud chapter.

  • External Id

    An unique id, that is used to access this Word Cloud via the API. Using this id, the Word Cloud can be used by external applications, such as 2Ring Gadgets. The External Id has to be unique across the entire Business Unit.

  • Calculation

    For calculation options, see Editing Table-Valued Calculations chapter.
    The calculation options control, how the displayed values are calculated.

    • Use Shared Calculation

      Whether shared calculation should be used for generating results.

    • Shared Calculation

      Available only if “Use Shared Calculation” is enabled
      A shared calculation to be used.

    • Text Column

      A column from the source table, from which the text will be used.

    • Weight Column

      A column from the source table that indicates the weight of the words.

  • Appearance

    The style of the Word Cloud.

    • Text Style

      Inheritable
      For more information see Text Style section.
      Word Cloud does not support Custom CSS in the Text Style. If Theme Text Style, with Custom CSS is chosen, Custom CSS will not be applied
      Type of font to use and whether bold or italic text should be used.

    • Spacing

      A gap between words.

    • Out Of Bound Behavior

      Inheritable
      Dictates what should happen with the word if it exceeds the source boundaries.

      Available options are:

      Hide - hides the word that exceeds the Word Cloud boundaries.

      If there is any word hidden due to exceeding the Word Cloud boundaries, the user will be notified in a pop up. Pop up appears only during editing of layout.

      Shrink To Fit - reduces the size of a word that exceeds the Word Cloud boundaries to fit into them.

      If there is a size reduction in any word due to exceeding the Word Cloud boundaries, the user will be notified in a pop up. Pop up appears only during editing of layout.

    • Word Sizing

      Word size options.

      • Size Parameter

        Inheritable
        Parameter on the basis of which the word size is calculated.

        Available options are: Weight and Weight Order:

        Weight - the size of the word depends on the weight assigned to the word.
        Weight Order - the size of the word is calculated on the basis of the order in which the word is dependent on the weight assigned to the word.

      • Manual Font Size Coefficients

        Whether the word size coefficients should be entered manually.

      • Minimum Font Size Coefficient

        Inheritable
        Available only if “Manual Font Size Coefficient” is enabled
        Minimum font size coefficient expresses relation of text size to height of the source. For example the value of 0.5 makes the header text half the size of the source. The font size will never drop below minimum size set.

      • Maximum Font Size Coefficient

        Inheritable
        Available only if “Manual Font Size Coefficient” is enabled
        Maximum font size coefficient expresses relation of text size to height of the source. For example the value of 0.5 makes the text half the size of the source. The font size will never exceed the maximum size set.

      • Sizing Function

        The function by which the word size is calculated.

        Available options are:

        Parabolic - differences in word size change parabolically, depending on the weight of the word
        Exponential - differences in word size change exponentialy, depending on the weight of the word. Differences in size of the words raises quicker than by using a parabolic function.

      • Parabolic Function Exponent

        Available only if “Parabolic” Sizing Function is selected
        Exponent, according to which the word size is calculated. The higher the value, the greater the difference between word sizes.

      • Exponential Function Base

        Available only if “Exponential” Sizing Function is selected
        A value, according to which the word size is calculated. The higher the value, the greater the difference between word sizes.

    • Word Rotation

      Word rotation options.

      • Minimum Rotation Angle

        Inheritable
        Minimum angle for word rotation in degrees. Word rotation will never be at a lower angle than a value set.

      • Maximum Rotation Angle

        Inheritable
        Maximum angle for word rotation in degrees. Word rotation will never exceed this angle.

      • Rotation Steps

        Inheritable
        The number of different angles of rotation of the word to be used.

      • Rotate Probability

        Inheritable
        The probability od words rotation in percent.

    • Colors

      Color options.

      • Color Mode

        Inheritable
        Available options are:

        Random - Randomly selcted color for each word.
        Columns - Every word is colored according to a hexadecimal value assigned to a word in a selected column.
        Fixed - A fixed color for all words.

      • Text Color

        Available only if “Fixed” Color Mode is selected
        For more information see Color Field section.
        Color of the words.

      • Color Column

        Available only if “Columns” Color Mode is selected
        A column, according to which the words will be colored.

      • Background Color

        Inheritable
        For more information see Color Field section.
        Color of the Word Cloud’s background.
