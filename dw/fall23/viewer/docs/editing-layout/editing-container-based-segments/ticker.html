Ticker¶

In Ticker, all items are displayed in a line. The line is automatically scrolling in a configured direction. When all items are scrolled through, the cycle starts again with the first item. This results in a perpetual showcase of all configured items.

For general information about this segment, see Ticker chapter.

  • Appearance

    The style of a Ticker

    • Scroll Speed

      Required
      Speed, at which the items are scrolled through.

    • Custom Scroll Speed

      Slider, adjusting the scroll speed with a text field for a custom value.
      Available only if Custom is selected as a Scroll Speed

    • Direction

      Required
      Direction, which the Ticker is scrolling.
      Available options are Right To Left, Left To Right, Bottom To Top and Top To Bottom

    • Background Color

      Inheritable
      Available only when Show Margins is enabled.
      For more information see Color field section.
      Color of the Ticker’s background.

  • Items

    Manually orderable
    List, displaying all items contained by the Ticker. A corresponding icon is displayed in front of each item, representing its type. An item without a selected type is represented by a question mark icon.

    Items are displayed in the order, determined by the Items list. The list is orderable, meaning the order can be manually changed.
