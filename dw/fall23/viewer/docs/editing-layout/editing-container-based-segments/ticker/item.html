Item¶

  • Type

    Required
    Determines, what type of Segment is displayed by the Item.

    Ticker, Sequence and Nested Layout are not supported as Ticker Items.

    When the type of a Ticker Item is selected, a second Section is displayed. This Section contains all options related to the selected type.

  • Title

    Title of the Item used in the Layout preview and also in the items list.

  • Width/Height

    Required
    Controls the corresponding size of the Item. The size is measured in permilles of the entire Layout. Only one of the fields is displayed at a time, depending on the scrolling direction.

  • Appearance

    The style of the Ticker’s item.

    • Corner Radius

      Inheritable
      Corner radius of the ticker’s item. can be adjusted with a slider or a text field for a custom value. Measured in permilles of the entire Layout’s height.

    • Margin

      Inheritable
      Margin of the ticker’s item can be adjusted with a slider or a text field for a custom value. Measured in permilles of the entire Layout’s height.

    • Shadow

      Inheritable
      For more information see Shadow Style section.
      Shadow of the ticker’s item.

  • Title and Paging Info

    Settings for the title and paging info.

    • Visibility

      Inheritable
      Whether the title should be visible.

    • Content

      Inheritable
      Content of the title.
      Available options are Title, Paging Info and Title and Paging Info

    • Position

      Inheritable
      Position of the title. If Float on Top or Float on Bottom is selected, title floats over the segment which may cause overlapping.
      Available options are Top, Bottom, Left and Right

    • Height

      Inheritable
      Height of the title, that is measured in permilles of the entire Layout’s height.

    • Text Color

      Inheritable
      For more information see Color Field section.
      Color of the text.

    • Background Color

      Inheritable
      For more information see Color Field section.
      Color of the background.

    • Text Style

      Inheritable
      For more information see Text Style section.
      Type of font to use and whether bold or italic text should be used.

    • Font Size Coefficient

      Inheritable
      The font size is calculated by multiplying available height for the text by font size coefficient.

    • Horizontal Alignment

      Available only if “Top” or “Bottom” is selected as a position
      Inheritable
      Position of the text on horizontal axis.
      Available options are Left, Center and Right

    • Padding

      Available only if “Top” or “Bottom” is selected as a position and “Left” or “Right” is selected as horizontal alignment.
      Inheritable
      The size of a padding between the Title text and Title’s border in “em” units. 1 em equals to the font size of the associated text.
      The padding is applied to the side where the text is horizontally aligned.

An item with Banner type selected is automatically resized to display its whole text. Therefore the Width/Height field is not present.

To simplify the creation of identical or similar Items, copy functionality is provided. Any existing Items can be copied. Once copied, the Item can then be pasted into other Layout. Copying Items behaves the same way as copying Layout Segment therefore see Layout Segment’s Copying chapter for more information.
