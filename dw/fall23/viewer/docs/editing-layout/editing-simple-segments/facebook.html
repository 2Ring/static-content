Social Feed - Facebook¶

Social Feed - Facebook segment provides ability to display any Facebook page directly in the DW segment.
It is not required to be logged in the Facebook to display page content.

For general information about this segment, see Social Feed - Facebook chapter.

In Google Chrome, facebook page may be rendered slightly blurred. “Auto Scale” setting located in the Advanced group can make the page more sharp in some cases. Other browsers like Edge or Firefox does not blur pages at all.

  • Page

    Required
    A Facebook page, that should be displayed. Use only the name of the page, for example: “Google”.

  • Hide Cover Photo

    Whether a cover photo of the page should be hidden.

  • Use Small Header

    Whether small header should be displayed.

  • Locale

    Localization of the page content.

  • Advanced

    Advanced settings.

    • Auto Scale

      Whether scaling of the page content should be automatically adjusted.

    • Scale

      Only available if “Auto Scale” is disabled.
      Controls the scaling of the content in the segment. This setting ensures that user can manually adjust the scaling for optimal visibility and readability of the page content.

    • Refresh Interval

      Required
      The page will be reloaded at a given interval.

    • Frames Swap Timeout

      Required
      Number of seconds the page takes to fully load. This is used to minimize flickering when page is being refreshed.
