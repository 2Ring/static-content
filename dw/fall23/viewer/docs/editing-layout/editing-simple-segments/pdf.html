PDF Document¶

The PDF Document Segment displays a PDF document, that can be automatically paged through. The document has to be saved on the server, where the DW application is deployed.

For general information about this segment, see PDF Document chapter.

  • File

    Required
    For more information see Static content chapter.
    Relative path of the PDF document saved on the server.

  • Reload Interval

    Required
    Interval, at which the document is checked for a newer version. If the document has been changed, the Segment is reloaded and paging resets. Otherwise no reload occurs and paging continues. If set to 0, document is loaded only once, when the Layout is navigated to.

  • Paging Interval

    Required
    Interval, at which the document is paged through.

  • Background Color

    Inheritable
    For more information see Color field section.
    Color of the PDF Document’s background.
