Shape¶

A Shape is a simple source, allowing user to create a rectagular, eliptical or triangular shapes with an option to change the color and add borders.

For general information about this segment, see Shape chapter.

  • Type

    Required
    Type of the shape.
    Available options are: Rectangle, Ellipse and Triangle

  • Background Color

    Required
    For more information see Color Field section.
    Color of the shapes background.

  • Border

    For more information see Borders Style section.
    The style, width and color settings for the shapes border.

  • Direction

    Only if “Triangle” is set as a Type
    The direction in which the triangle is rotated.
    Available options are: Left, Right, Top and Bottom
