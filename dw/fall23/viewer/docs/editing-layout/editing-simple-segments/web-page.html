Web Page¶

An external website or webpage can be displayed inside of the Layout, using the Web Page Segment type. The website is fully interactable in the Layout view.

For general information about this segment, see Web Page chapter.

Some websites don’t support embedding and cannot be used in the Web Page segment. When such website is selected, an empty page with broken file icon is displayed instead.

  • URL

    Required
    Address of the website, that should be displayed. http:// or https:// has to be included.

  • Refresh Interval

    Required
    If set to 0, the Web Page is loaded only once, when the Layout is navigated to. Otherwise, the page will be reloaded at a given interval.

  • Background Color

    Inheritable
    For more information see Color field section.
    Color of the Web Page’s background in case page is transparent.
