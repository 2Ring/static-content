What is a LayoutÂ¶

A Layout is a key component of the DW application. Each Layout can be configured to display key information in form of a wallboard. Layout Segments are the primary parts of a Layout. Many different types of segments can be used, each serving a specific purpose. A Layout is always associated with a Business Unit (BU). A Layout can be seen by any user, that is part of its BU.

The common theme for the entire application is customizable as well. Colors, fonts and more can be changed to match the theme of the company. Furthermore, style of each segment can be individually set.
