Even though the basic purpose of both the Layout and the Widget is the same, to display a key information in form of a wallboard, the Widget offers a much more easier solution for displaying the same statistics for a multiple agents or teams. Data which are displayed in Widget can be parametrized by layout where widget is used thus eliminating the need to create specific widget for each agent or team.
