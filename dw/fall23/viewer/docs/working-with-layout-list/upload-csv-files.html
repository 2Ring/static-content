Upload CSV FilesÂ¶

CSV files can be uploaded and then selected as a source table in table valued calculations. This type of calculation provides a static data that are not updated.

First column in the CSV file is considered as a key, therefore it should contain unique values that serve as a specific identifier for example, for an agent, queue or team.

To upload a CSV file, click the Upload CSV File button in the layout list.

Figure 1: Upload CSV File button

In the static content dialog, upload the CSV file to a desired Business Unit. For more information about static content management, see Content management chapter.

A CSV file can only be selected as a source table within the business unit to which it was uploaded or if it was uploaded to shared business unit.
