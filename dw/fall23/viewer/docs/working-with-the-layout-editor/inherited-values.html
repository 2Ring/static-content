Inherited Values¶

Inherited values are values set on a higher level entity that are automatically applied if user does not provide other value. These values are displayed in cursive style and light grey color. The source of the inheritance is shown in brackets. Different types of input fields display the inherited value as follows:

Figure 1: Fields with inheritable values

  1. Value and color fields

    Field’s text area displays the inherited value when no user-specified value has been set. Leaving the field in this state will automatically use the inherited value.

  2. Dropdowns

    The inherited value is added to the list of the dropdown’s options. Selecting this option will use the inherited value.
