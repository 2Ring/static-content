Keyboard Shortcuts¶

While working with the Layout Editor, keyboard shortcuts can be used to perform common actions.

The following keyboard shortcuts are available:

Table 1 Keyboard Shortcuts¶

Shortcut

Action

Notes

Ctrl + a

Select all segments in the layout

Layout’s edit panel has to be focused

Ctrl + c

Put selected items into the clipboard

Layout’s edit panel or the segment’s edit panel has to be focused

Ctrl + v

Copies items from the clipboard

Layout’s edit panel or the segment’s edit panel has to be focused

Delete

Delete selected items

Layout’s edit panel or the segment’s edit panel has to be focused

Ctrl + Left Mouse Button

Add or remove segments from the selection in the layout

Segments are moved simultaneously in the layout in case of moving a group of segments.

Multiple segments can be selected in the layout editor also by clicking and dragging the cursor over the desired segments.
