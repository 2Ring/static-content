Lists¶

Figure 1: List

Lists display any collection of items. These items can be managed using the following controls:

  1. Search field

    Enter search criteria to quickly filter displayed items.

  2. Select/Deselect all items checkbox

    Click to select/deselect all items.

  3. Item selection checkbox

    Click to select/deselect a single item.

  4. Add button

    Click to add new items.

  5. Copy button

    Available only if at least one item is selected.
    Click to copy selected items.

  6. Paste button

    Available only if at least one item is in the clipboard.
    Click to paste copied items.

  7. Delete button

    Click to delete selected items. This action is , if at least one item is selected.

Clicking item or adding an item, opens up a new panel, containing all its related configuration options.

The name of the item is determined by its most distinctive option, such as Title. When not set, (Untitled) is displayed instead.

By default, the list’s items are sorted alphabetically in ascending order. Some Lists are manually orderable by dragging items to their desired position.
